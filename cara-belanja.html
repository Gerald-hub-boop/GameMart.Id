Cara Mudah Belanja di GameMart.ID

Ikuti 5 langkah simpel ini untuk mendapatkan item game favoritmu!

LANGKAH 1

Pilih Game dan Produk

Cari game favoritmu melalui halaman utama atau gunakan kolom pencarian. Setelah itu, pilih item atau nominal top up yang kamu inginkan (contoh: 257 Diamonds).

LANGKAH 2

Masukkan User ID Game

Lengkapi data yang dibutuhkan, seperti User ID dan Zone ID (jika ada). Pastikan data yang kamu masukkan sudah benar untuk menghindari kesalahan pengiriman.

LANGKAH 3

Pilih Metode Pembayaran

Pilih metode pembayaran yang paling nyaman untukmu. Kami menyediakan E-Wallet, Transfer Bank (Virtual Account), dan QRIS yang bisa dipindai oleh semua aplikasi pembayaran.

LANGKAH 4

Selesaikan Pembayaran

Lakukan pembayaran sesuai instruksi yang tertera di layar. Sistem akan memverifikasi pembayaranmu secara otomatis.

LANGKAH 5

Item Langsung Masuk!

Selamat! Item akan langsung dikirim ke akun game kamu dalam hitungan detik. Selamat bermain!
